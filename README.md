# ML-Models
This repository contains my machine learning and analytics projects built while exploring practical applications of data science across business, consumer behaviour, forecasting, and decision-making problems.

My current interests include:

Machine Learning
Predictive Analytics
Time-Series Forecasting
Recommendation & Segmentation Models
AI-assisted Development Workflows
Business and Product Analytics

Most projects here are built using Python, Pandas, Scikit-learn, and visualization libraries, with increasing exploration into deep learning and AI systems.

I use these projects as a way to learn by building — experimenting with models, understanding their assumptions, interpreting outputs, and translating technical findings into business insights.

Currently on a continuous journey of learning more about ML, AI, product thinking, and data-driven strategy.
